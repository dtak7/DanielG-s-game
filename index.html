UFO Survivor

by: Daniel Goldstein

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

This maze game was the kind of game that first came to mind. I didn't really know what else to do so I made this game.

2. What I Learned

I learned that even though things might seem clear to you, other people might not understand what is going on. I made the game instructions more specific, and made the game easier.

3. Architecture of My Game

The different aspects of the game are all in their own classes, which are then used in the game panel. The game panel is then called in the runner class.

4. Technical Implementation

I am proud of the giant green laser that comes in if time runs out, because it taught me how to use timers.

5. Most Challenging Part

The timer sped up for random reasons and it took awhile to debug the project.

6. Future Fixes and Updates

I would maybe add a storyline and also make a different maze.
